# 台灣旅遊景點導覽 | [Demo](https://johnsonmao.github.io/taiwan_tourguide/#/home)

## Introduction

使用 React 串接 [TDX 運輸資料流通服務](https://tdx.transportdata.tw/) API，實作台灣旅遊景點導覽功能。

## Features

- [x] 關鍵字篩選功能
- [x] 定位搜索附近功能
- [x] 依地區分類功能
- [ ] RWD 手機功能優化（開發中）
- [ ] 交通資訊功能（開發中）
